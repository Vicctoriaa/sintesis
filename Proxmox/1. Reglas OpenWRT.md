## Configuración de red (UCI)

```bash
# WAN — IP estática
uci set network.wan.device='eth0'
uci set network.wan.proto='static'
uci set network.wan.ipaddr='192.168.3.200'
uci set network.wan.netmask='255.255.255.0'
uci set network.wan.gateway='192.168.3.1'

# VLAN 10 — DMZ
uci set network.vlan10=interface
uci set network.vlan10.device='eth1.10'
uci set network.vlan10.proto='static'
uci set network.vlan10.ipaddr='10.1.1.1'
uci set network.vlan10.netmask='255.255.255.224'

# VLAN 20 — Servicios
uci set network.vlan20=interface
uci set network.vlan20.device='eth1.20'
uci set network.vlan20.proto='static'
uci set network.vlan20.ipaddr='10.1.1.33'
uci set network.vlan20.netmask='255.255.255.224'

# VLAN 30 — SOC
uci set network.vlan30=interface
uci set network.vlan30.device='eth1.30'
uci set network.vlan30.proto='static'
uci set network.vlan30.ipaddr='10.1.1.65'
uci set network.vlan30.netmask='255.255.255.224'

# VLAN 40 — Producción
uci set network.vlan40=interface
uci set network.vlan40.device='eth1.40'
uci set network.vlan40.proto='static'
uci set network.vlan40.ipaddr='10.1.1.97'
uci set network.vlan40.netmask='255.255.255.224'

# VLAN 50 — ???
uci set network.vlan50=interface
uci set network.vlan50.device='eth1.50'
uci set network.vlan50.proto='static'
uci set network.vlan50.ipaddr='10.1.1.129'
uci set network.vlan50.netmask='255.255.255.224'

uci commit network
service network restart
```

## Firewall

### Política por defecto

| Dirección | Política |
|-----------|----------|
| Input global | REJECT |
| Output global | ACCEPT |
| Forward global | REJECT |
| SYN flood protection | Activo |

### Zonas

| Zona | Red | Input | Output | Forward | Masquerade |
|------|-----|-------|--------|---------|------------|
| wan | eth0 | REJECT | ACCEPT | REJECT | Sí |
| vlan10 | eth1.10 | ACCEPT | ACCEPT | REJECT | No |
| vlan20 | eth1.20 | ACCEPT | ACCEPT | REJECT | No |
| vlan30 | eth1.30 | ACCEPT | ACCEPT | REJECT | No |
| vlan40 | eth1.40 | ACCEPT | ACCEPT | REJECT | No |
| vlan50 | eth1.50 | ACCEPT | ACCEPT | REJECT | No |

### Forwardings (tráfico permitido entre zonas)

| Origen | Destino | Descripción |
|--------|---------|-------------|
| vlan10 | wan | DMZ → Internet |
| vlan20 | wan | Servicios → Internet |
| vlan30 | wan | SOC → Internet |
| vlan40 | wan | Producción → Internet |
| vlan50 | wan | ??? → Internet |

> El tráfico entre VLANs está bloqueado (`forward=REJECT`). Ninguna VLAN puede acceder a otra directamente.

### Reglas activas

| Nombre | Origen | Protocolo | Puerto | Acción |
|--------|--------|-----------|--------|--------|
| Allow-DHCP-Renew | wan | UDP | 68 | ACCEPT |
| Allow-Ping | wan | ICMP | echo-request | ACCEPT |
| Allow-IGMP | wan | IGMP | — | ACCEPT |
| Allow-DHCPv6 | wan | UDP | 546 | ACCEPT |
| Allow-MLD | wan | ICMPv6 | 130-143 | ACCEPT |
| Allow-ICMPv6-Input | wan | ICMPv6 | varios | ACCEPT |
| Allow-ICMPv6-Forward | wan | ICMPv6 | varios | ACCEPT |

### Comandos para aplicar el firewall

```bash
# Zonas
uci add firewall zone
uci set firewall.@zone[-1].name='vlan10'
uci set firewall.@zone[-1].network='vlan10'
uci set firewall.@zone[-1].input='ACCEPT'
uci set firewall.@zone[-1].output='ACCEPT'
uci set firewall.@zone[-1].forward='REJECT'

# (repetir para vlan20, vlan30, vlan40, vlan50)

# Forwardings
uci add firewall forwarding
uci set firewall.@forwarding[-1].src='vlan10'
uci set firewall.@forwarding[-1].dest='wan'

# (repetir para vlan20, vlan30, vlan40, vlan50)

uci commit firewall
service firewall restart
```
