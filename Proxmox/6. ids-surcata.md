# Suricata — Configuración CT 106

## Datos del contenedor

La siguiente tabla resume los parámetros principales del contenedor LXC donde se despliega Suricata. Cada campo define un aspecto del entorno virtualizado en Proxmox.

| Campo | Valor |
|-------|-------|
| CT ID | 106 |
| Hostname | suricata-ids |
| Imagen | debian-12-standard_12.12-1_amd64.tar.zst |
| Memoria | 1024 MB |
| Disco | 8 GB (local-lvm) |
| Cores | 2 |
| Privilegiado | Sí |


⚙️ 1. Instalación

En tu LXC (Debian/Ubuntu):

apt update
apt install jq -y
🚀 2. Después de instalarlo

Ya podrás hacer esto:

🔥 Solo alertas reales:
jq 'select(.event_type=="alert")' /var/log/suricata/eve.json

🔥 Solo eventos HTTP:
jq 'select(.event_type=="http")' /var/log/suricata/eve.json

🔥 Solo tráfico sospechoso:
jq 'select(.event_type=="alert" or .event_type=="anomaly")' /var/log/suricata/eve.json
