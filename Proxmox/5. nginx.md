## Nginx

### Instalación

```bash
apt update && apt install -y nginx
```

### Virtual host Grafana `/etc/nginx/sites-available/grafana`

```nginx
server {
    listen 3000;
    server_name _;

    location / {
        proxy_pass http://10.1.1.66:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;

        # WebSockets para Grafana Live
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

### Virtual host SOAR `/etc/nginx/sites-available/soar`

```nginx
server {
    listen 8080;
    server_name _;

    location / {
        proxy_pass http://10.1.1.37:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### Virtual host Web (reservado) `/etc/nginx/sites-available/web`

```nginx
server {
    listen 80 default_server;
    server_name _;

    location / {
        return 503 "Próximamente";
    }
}
```

### Activar y recargar

```bash
# Limpiar habilitados anteriores
rm /etc/nginx/sites-enabled/*

# Activar los nuevos
ln -s /etc/nginx/sites-available/grafana /etc/nginx/sites-enabled/
ln -s /etc/nginx/sites-available/soar /etc/nginx/sites-enabled/
ln -s /etc/nginx/sites-available/web /etc/nginx/sites-enabled/

# Verificar y recargar
nginx -t && systemctl reload nginx
```

## Port forwarding en Proxmox

Reglas PREROUTING en el host `honeycos` para redirigir tráfico externo a CT105:

```bash
iptables -t nat -A PREROUTING -i vmbr0 -p tcp --dport 3000 -j DNAT --to 10.1.1.35:3000
iptables -t nat -A PREROUTING -i vmbr0 -p tcp --dport 8080 -j DNAT --to 10.1.1.35:8080
```

> Wazuh (puerto 443) tiene su propia regla directa a VM202, sin pasar por Nginx:
> ```bash
> iptables -t nat -A PREROUTING -i vmbr0 -p tcp --dport 443 -j DNAT --to 10.1.1.67:443
> ```

### Flujo de acceso

```
Cliente (192.168.3.x)
   ↓ http://192.168.3.200:3000
Proxmox vmbr0 (NAT :3000 → 10.1.1.35:3000)
   ↓
CT105 Nginx (10.1.1.35:3000)
   ↓ proxy_pass
CT101 Grafana (10.1.1.66:3000)

Cliente (192.168.3.x)
   ↓ http://192.168.3.200:8080
Proxmox vmbr0 (NAT :8080 → 10.1.1.35:8080)
   ↓
CT105 Nginx (10.1.1.35:8080)
   ↓ proxy_pass
CT104 SOAR (10.1.1.37:8080)

Cliente (192.168.3.x)
   ↓ https://192.168.3.200:443
Proxmox vmbr0 (NAT :443 → 10.1.1.67:443)
   ↓ directo (sin Nginx)
VM202 Wazuh Dashboard (10.1.1.67:443)
```
