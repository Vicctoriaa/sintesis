# Sistema de Backups Proxmox VE → Ubuntu Server con rsync

## Descripción

Este proyecto implementa un sistema de backups automatizado para un entorno de virtualización basado en **Proxmox VE**, enviando copias de seguridad a un **Ubuntu Server** mediante `rsync` sobre SSH.

El objetivo es disponer de una solución:

* Open Source
* Funcional con un solo disco
* Automatizada
* Incremental
* Segura (SSH)

---

## Arquitectura

```
[ Proxmox VE ]
     │
     │ vzdump (snapshots/backups)
     ▼
/var/lib/vz/dump/
     │
     │ rsync (SSH)
     ▼
[ Ubuntu Server ]
/backups/proxmox/
```

---

## Requisitos

### En Proxmox VE

* Acceso root
* `rsync` instalado (normalmente ya viene)
* Backups habilitados (`vzdump`)

### En Ubuntu Server

* SSH instalado (`openssh-server`)
* Usuario para backups
* Espacio disponible en disco

---

## Configuración

### Preparar Ubuntu Server

```bash
sudo apt update
sudo apt install openssh-server -y
sudo systemctl enable ssh
sudo systemctl start ssh
```

### Crear usuario de backup

```bash
sudo adduser backupuser
sudo mkdir -p /backups/proxmox
sudo chown backupuser:backupuser /backups/proxmox
sudo chmod 700 /backups/proxmox
```

---

### Configurar acceso SSH sin contraseña

En Proxmox VE:

```bash
ssh-keygen
ssh-copy-id backupuser@IP_UBUNTU
```

---

## Script de Backup

Crear el archivo:

```bash
nano /root/proxmox_backup.sh
```

### Contenido del script

```bash
#!/bin/bash

# ==============================
# CONFIGURACIÓN
# ==============================

DESTINO="backupuser@IP_UBUNTU:/backups/proxmox"
ORIGEN="/var/lib/vz/dump/"
LOG="/var/log/proxmox_backup.log"
FECHA=$(date +"%Y-%m-%d %H:%M:%S")

echo "====================================" >> $LOG
echo "Inicio backup: $FECHA" >> $LOG

# ==============================
# BACKUP DE VMs
# ==============================

echo "Backup de VMs..." >> $LOG
for VMID in $(qm list | awk 'NR>1 {print $1}'); do
    echo "→ VM $VMID" >> $LOG
    vzdump $VMID --mode snapshot --compress lzo --storage local >> $LOG 2>&1
done

# ==============================
# BACKUP DE CONTENEDORES
# ==============================

echo "Backup de CTs..." >> $LOG
for CTID in $(pct list | awk 'NR>1 {print $1}'); do
    echo "→ CT $CTID" >> $LOG
    vzdump $CTID --mode snapshot --compress lzo --storage local >> $LOG 2>&1
done

# ==============================
# SINCRONIZACIÓN CON RSYNC
# ==============================

echo "Sincronizando con servidor remoto..." >> $LOG

rsync -avh --delete $ORIGEN $DESTINO >> $LOG 2>&1

# ==============================
# FIN
# ==============================

echo "Backup finalizado: $(date +"%Y-%m-%d %H:%M:%S")" >> $LOG
```

---

### Dar permisos de ejecución

```bash
chmod +x /root/proxmox_backup.sh
```

---

## Automatización con Cron

Editar tareas programadas:

```bash
crontab -e
```

Ejemplo: ejecutar cada día a las 2 AM

```bash
0 2 * * * /root/proxmox_backup.sh
```
