## Interfaces de red

| Interfaz | Bridge Proxmox | Rol | IP |
|----------|---------------|-----|----|
| eth0 | vmbr0 | WAN (red externa) | 192.168.3.109/24 (DHCP) |
| eth1 | vmbr1 | LAN (red interna SOC) | 10.1.1.1/24 (estática) |

### Esquema

```
Internet
   ↓
Router (192.168.3.1)
   ↓
Proxmox vmbr0 (192.168.3.200)
   ↓
OpenWRT CT100
   ├── eth0 → 192.168.3.109/24  (WAN)

                  ↓
           vmbr1 (10.1.1.0/24)
           └── CTs/VMs SOC
```

---

## Configuración de red (UCI)

```bash
# WAN
uci set network.wan.device='eth0'
uci set network.wan.proto='dhcp'

# LAN
uci set network.lan.device='eth1'
uci set network.lan.ipaddr='10.1.1.1'
uci set network.lan.netmask='255.255.255.0'

uci commit network
/etc/init.d/network restart
```

## DHCP

### Configuración aplicada

| Parámetro | Valor |
|-----------|-------|
| Interfaz | eth1 (LAN) |
| Rango | 10.1.1.100 — 10.1.1.253 |
| Lease time | 12h |
| Gateway (opción 3) | 10.1.1.1 |
| DNS (opción 6) | 10.1.1.1 |
| DHCP en WAN | desactivado |

### Comandos aplicados

```bash
uci set dhcp.lan.start='100'
uci set dhcp.lan.limit='154'
uci set dhcp.lan.leasetime='12h'
uci delete dhcp.lan.dhcp_option
uci add_list dhcp.lan.dhcp_option='3,10.1.1.1'
uci add_list dhcp.lan.dhcp_option='6,10.1.1.1'
uci commit dhcp
/etc/init.d/dnsmasq restart
```

### Verificación

```bash
cat /var/etc/dnsmasq.conf.cfg01411c | grep -E "interface|dhcp-range"
# dhcp-range=set:lan,10.1.1.100,10.1.1.253,255.255.255.0,12h
# no-dhcp-interface=eth0
```

## Comandos útiles

```bash
# Entrar al CT desde Proxmox
pct enter 100

# Ver interfaces

# Ver leases DHCP activos
cat /tmp/dhcp.leases

# Reiniciar red
/etc/init.d/network restart

# Reiniciar DHCP/DNS
/etc/init.d/dnsmasq restart

# Ver configuración UCI completa
uci show network
uci show dhcp
```

## DNS apuntando al CT-103

Configurado dnsmasq para que toda la red interna use `10.1.1.5` (CT-103) como servidor DNS:

```bash
uci set dhcp.@dnsmasq[0].server='10.1.1.5'
uci set dhcp.@dnsmasq[0].noresolv='1'
uci commit dhcp
/etc/init.d/dnsmasq restart
```

### Verificación

```bash
cat /var/etc/dnsmasq.conf.cfg01411c | grep -E "server|interface|dhcp-range"
```
