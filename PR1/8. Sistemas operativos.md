# 🖥️ Sistemas Operativos Utilizados

En nuestra infraestructura utilizamos **Proxmox VE** como plataforma de virtualización. Este sistema está basado en **Debian Linux** y nos permite gestionar tanto **contenedores LXC** como **máquinas virtuales**, donde desplegaremos los diferentes servicios del laboratorio.

La mayoría de los contenedores utilizan **Debian Server** como sistema operativo base, ya que es una distribución **estable, ligera y muy utilizada en entornos de servidor** y compatible con muchas herramientas de seguridad.

## 📊 Resumen de Sistemas Operativos

| ID | Servicio | Tipo | Sistema Operativo |
|----|----------|------|------------------|
| CT100 | Firewall | Contenedor | OpenWRT |
| CT101 | IDS | Contenedor | Debian Server |
| CT102 | Grafana / Prometheus | Contenedor | Debian Server |
| CT103 | Playbooks / DNS | Contenedor | Debian Server |
| CT104 | SOAR Web | Contenedor | Debian Server |
| CT105 | Honeypot | Contenedor | Debian Server |
| VM200 | Windows Active Directory | Máquina Virtual | Windows Server |

## 🎯 Resumen

En nuestro laboratorio predominan los sistemas **basados en Linux**, principalmente **Debian Server**, por su estabilidad y bajo consumo de recursos.  
Además, utilizamos **OpenWRT** para el sistema de red y **Windows Server** para simular un entorno empresarial con Active Directory.
