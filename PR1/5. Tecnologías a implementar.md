## Tecnologías utilizadas

Las principales tecnologías utilizadas en el proyecto son:

### Proxmox VE
  - **Proxmox Virtual Environment (Proxmox VE)** es una plataforma de virtualización de código abierto que permite crear y administrar **máquinas virtuales y contenedores** desde una interfaz web.  
  En este proyecto se utiliza para **gestionar la infraestructura virtual donde se ejecutan los distintos servicios**.

### Docker
  - **Docker** Docker es una plataforma que sirve para crear, ejecutar y gestionar aplicaciones dentro de contenedores.
  Se utiliza para **desplegar y ejecutar servicios del proyecto en contenedores de forma aislada y segura**.

### OpenWRT
  - **OpenWRT** es un sistema operativo basado en Linux diseñado para **routers y dispositivos de red**.  
  En este proyecto se emplea para **gestionar la red, configurar reglas de firewall y controlar el tráfico de red**.

### Suricata
  - **Suricata** es un sistema de **detección y prevención de intrusiones (IDS/IPS)**.  
  Se utiliza para **analizar el tráfico de red en tiempo real y detectar posibles ataques o actividades sospechosas**.

### Grafana
  - **Grafana** es una plataforma de **visualización y análisis de datos** que permite crear dashboards interactivos.  
  En el proyecto se utiliza para **mostrar métricas del sistema y de la red de forma visual y en tiempo real**.

### Prometheus
  - **Prometheus** es un sistema de **monitorización y recolección de métricas**.  
  Se encarga de **recoger datos del sistema y enviarlos a Grafana para su visualización**.

### PHP
  - **PHP** es un lenguaje de programación utilizado principalmente para **desarrollo web del lado del servidor**.  
  Se utiliza en el proyecto para **desarrollar la lógica de la aplicación web**.

### MariaDB / MySQL
  - **MariaDB** y **MySQL** son sistemas de **gestión de bases de datos relacionales**.  
  Se emplean para **almacenar y gestionar la información de la aplicación**.

### Bash
  -  **Bash** es un intérprete de comandos utilizado en sistemas Linux.  
  Se usa para **automatizar tareas del sistema mediante scripts**.

### SMTP
  - **SMTP (Simple Mail Transfer Protocol)** es un protocolo utilizado para **enviar correos electrónicos**.  
  En el proyecto se utiliza para **enviar notificaciones o alertas automáticas**.

### LDAP
  - **LDAP** es un servicio de directorio desarrollado por Microsoft que permite **gestionar usuarios, permisos y recursos dentro de una red**.  
  Se utiliza para **centralizar la autenticación y la gestión de usuarios**.
