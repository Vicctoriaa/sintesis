## Tecnologías utilizadas

El proyecto integra diversas tecnologías organizadas en diferentes áreas funcionales: virtualización, red, seguridad, monitorización, desarrollo y servicios auxiliares.

---

## 1. Virtualización e infraestructura

### Proxmox VE
**Proxmox Virtual Environment (Proxmox VE)** es la plataforma principal de virtualización del proyecto.

Permite gestionar máquinas virtuales y contenedores LXC desde una interfaz centralizada, siendo la base de toda la infraestructura del laboratorio.

---

### Docker
**Docker** es una plataforma de contenedores que permite desplegar aplicaciones de forma aislada, ligera y escalable.

Se utiliza para ejecutar servicios de manera eficiente dentro del entorno virtualizado.

---

## 2. Redes y comunicaciones

### OpenWRT (Firewall)
**OpenWRT** es un sistema operativo basado en Linux orientado a dispositivos de red.

Se utiliza como firewall para:
- Control del tráfico de red
- Aplicación de reglas de seguridad
- Segmentación de la red

---

### DNS (Bind9)
**DNS (Domain Name System)** permite traducir nombres de dominio en direcciones IP.

En el proyecto se utiliza **Bind9** como servidor DNS interno.

---

### DHCP
**DHCP (Dynamic Host Configuration Protocol)** asigna automáticamente direcciones IP y configuración de red a los dispositivos.

---

### VLAN
**VLAN (Virtual Local Area Network)** permite segmentar una red física en múltiples redes lógicas independientes.

Se utiliza para mejorar la seguridad y organización del entorno.

---

### Reverse Proxy (Nginx)
Un **reverse proxy** actúa como intermediario entre los clientes y los servidores internos.

Se utiliza para:
- Mejorar la seguridad
- Gestionar el tráfico
- Optimizar el rendimiento

---

## 3. Seguridad informática

### Suricata
**Suricata** es un sistema IDS/IPS.

Permite:
- Detectar intrusiones
- Analizar tráfico en tiempo real
- Generar alertas de seguridad

---

### Wazuh
**Wazuh** es una plataforma SIEM/XDR.

Se utiliza para:
- Monitorización de seguridad
- Detección de amenazas
- Respuesta ante incidentes

---

### Vaultwarden
**Vaultwarden** es un gestor de contraseñas.

Permite almacenar credenciales de forma segura utilizando una contraseña maestra.

---

### Tailscale (VPN)
**Tailscale** es una solución VPN basada en WireGuard.

Permite la conexión segura entre dispositivos sin configuraciones complejas de red.

---

### Cloudflare Tunnel
**Cloudflare Tunnel** permite exponer servicios internos de forma segura sin necesidad de abrir puertos.

Proporciona:
- Acceso remoto seguro
- Protección contra ataques
- Integración con servicios en la nube

---

## 4. Monitorización y análisis

### Prometheus
**Prometheus** es un sistema de monitorización y recolección de métricas.

Recoge datos del sistema para su análisis.

---

### Grafana
**Grafana** es una herramienta de visualización de datos.

Permite crear dashboards interactivos para monitorizar el estado del sistema.

---

## 5. Desarrollo web y automatización

### PHP
**PHP** es un lenguaje de programación del lado del servidor.

Se utiliza para el desarrollo de la aplicación web del sistema SOAR.

---

### HTML
**HTML** es el lenguaje de marcado utilizado para estructurar páginas web.

---

### CSS
**CSS** se utiliza para definir el diseño y la apariencia visual de las páginas web.

---

### Bash (Playbooks)
**Bash** es un intérprete de comandos de Linux.

Se utiliza para automatizar tareas y crear scripts de respuesta ante incidentes.

---

## 6. Servicios de sistema y gestión

### MariaDB / MySQL
Sistemas de gestión de bases de datos relacionales.

Se utilizan para almacenar y gestionar la información del sistema.

---

### Postfix (Servidor de correo)
**Postfix** es un servidor de correo (MTA).

Se utiliza para el envío de alertas y notificaciones automáticas.

---

### SMTP
**SMTP (Simple Mail Transfer Protocol)** es el protocolo estándar para el envío de correos electrónicos.

---

### LDAP / Active Directory
Servicio de directorio utilizado para la gestión centralizada de usuarios, permisos y políticas de acceso en la red.

---
