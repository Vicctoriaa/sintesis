# 🧩 2. Listado de tareas

Para poder desarrollar el proyecto será necesario realizar una serie de tareas organizadas en diferentes fases.

Primero se realizará la instalación del entorno base, que incluye la instalación de **Proxmox** y la configuración inicial del servidor. Posteriormente se crearán los contenedores y máquinas virtuales necesarios para los distintos servicios.

Las principales tareas serán:

- [ ] Instalación y configuración de Proxmox
- [ ] Creación de contenedores y máquinas virtuales
- [ ] Configuración del firewall
- [ ] Instalación del IDS (Snort / Suricata)
- [ ] Configuración de Grafana y Prometheus
- [ ] Desarrollo del SOAR Web
- [ ] Implementación de playbooks
- [ ] Configuración de Active Directory
- [ ] Implementación del honeypot
- [ ] Realización de pruebas
- [ ] Documentación final
