# 🧩 2. Listado de tareas

Para poder desarrollar el proyecto será necesario realizar una serie de tareas organizadas en diferentes fases.

Primero se realizará la instalación del entorno base, que incluye la instalación de **Proxmox** y la configuración inicial del servidor. Posteriormente se crearán los contenedores y máquinas virtuales necesarios para los distintos servicios.

Las principales tareas serán:

- Instalación y configuración del servidor **Proxmox VE**.
- Creación de contenedores LXC y máquinas virtuales.
- Configuración de la red interna del laboratorio.
- Instalación del **firewall**.
- Configuración del **IDS (Snort o Suricata)**.
- Instalación de herramientas de monitorización como **Grafana y Prometheus**.
- Desarrollo de la plataforma web **SOAR**.
- Implementación de scripts de automatización (**playbooks**).
- Configuración de **Active Directory** en Windows Server.
- Implementación del **honeypot**.
- Realización de pruebas de funcionamiento.
- Documentación final del proyecto.
