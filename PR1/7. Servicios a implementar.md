# 7. Servicios a implementar

Dentro de la infraestructura se desplegarán los siguientes servicios:

| Servicio | Descripción |
|--------|-------------|
| Proxmox | Hipervisor tipo 1 |
| Firewall | Control del tráfico de red, seguridad y manejo de la red interna |
| IDS | Detección de intrusiones |
| Monitorización | Visualización de métricas |
| SOAR Web | Página web |
| Playbooks | Automatización de respuestas (Scripts) |
| Active Directory | Simulación de entorno empresarial |
| Honeypot | Sistema señuelo para detectar ataques |
| Proxmox backup | Servidor para copias de seguridad |
| TrueNAS | Para copias de seguridad, conectado al Proxmox backup |
