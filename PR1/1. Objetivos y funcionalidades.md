# 🎯 1. Objetivos y funcionalidades

El objetivo principal del proyecto es diseñar e implementar un entorno que simule la infraestructura de seguridad de una empresa. Para ello se desplegarán diferentes servicios que permitan detectar actividad sospechosa dentro de la red.

Los objetivos específicos del sistema son:

- Implementar una infraestructura virtualizada utilizando **Proxmox VE**.
- Monitorizar la red y los sistemas mediante herramientas de análisis y métricas.
- Detectar posibles intrusiones mediante un **IDS (Intrusion Detection System)**.
- Visualizar los datos de seguridad mediante dashboards.
- Automatizar respuestas ante incidentes mediante scripts o playbooks.
- Simular un entorno empresarial mediante **Active Directory**.
- Implementar un **honeypot** para atraer y analizar posibles ataques.

Gracias a estas funcionalidades se podrá observar el comportamiento de posibles intrusos y analizar los eventos generados dentro del sistema.
