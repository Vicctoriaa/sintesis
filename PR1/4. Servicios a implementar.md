## 4. Servicios a implementar

Dentro de la infraestructura se han desplegado distintos servicios, cada uno con un rol específico dentro del entorno de ciberseguridad, monitorización y gestión del sistema.

| Servicio | Descripción |
|----------|-------------|
| **Proxmox VE** | Hipervisor de tipo 1 encargado de la virtualización y gestión de máquinas virtuales y contenedores |
| **Firewall** | Control del tráfico de red, segmentación, filtrado de paquetes y protección de la red interna |
| **IDS** | Sistema de detección de intrusiones encargado de monitorizar el tráfico y detectar actividad maliciosa |
| **Monitorización / SIEM** | Plataforma para la recolección, análisis y visualización de métricas y eventos de seguridad |
| **SOAR Web** | Aplicación web para la gestión centralizada de alertas y automatización de respuestas |
| **Playbooks** | Scripts de automatización para la respuesta ante incidentes de seguridad |
| **Directorio de dominio (LDAP)** | Servicio de autenticación y gestión centralizada de usuarios y políticas en un entorno empresarial simulado |
| **Honeypot** | Sistema señuelo diseñado para atraer, registrar y analizar intentos de ataque |
| **Backup** | Servicio dedicado a la realización y gestión de copias de seguridad del sistema |
| **Servidor de correo** | Sistema para el envío de alertas, notificaciones y logs del sistema |
| **Servidor VPN** | Servicio de acceso remoto seguro a la infraestructura del laboratorio |
| **Servidor DNS** | Servicio de resolución de nombres de dominio a direcciones IP dentro de la red |
| **Servidor Proxy** | Servidor intermediario que gestiona y filtra las conexiones entre cliente y servidor destino, controlando el tráfico de red |
