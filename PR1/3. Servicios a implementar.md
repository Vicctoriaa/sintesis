# 7. Servicios a implementar 

Dentro de la infraestructura se desplegarán los siguientes servicios:

| Servicio | Descripción |
|--------|-------------|
| Proxmox | Hipervisor tipo 1 |
| Firewall | Control del tráfico de red, seguridad y manejo de la red interna |
| IDS | Detección de intrusiones |
| Monitorización - SIEM | Visualización de métricas |
| SOAR Web | Página web |
| Playbooks | Automatización de respuestas (Scripts) |
| Directorio de dominio| Simulación de entorno empresarial |
| Honeypot | Sistema señuelo para detectar ataques |
| Backup | Servidor para copias de seguridad |
| Servidor de correo | Servidor para el envío de alertas y logs |
| Servidor VPN | Servidor dedicado para la conexión remotada hacía nuestro servidor |
| Servidor DNS |  Servidor para traducir nombres de dominio legibles por humanos en direcciones IP que las máquinas pueden entender |
| Servidor Proxy |Servidor que actúa como intermediario terminando la conexión TCP del cliente y estableciendo una nueva hacia el servidor destino |
