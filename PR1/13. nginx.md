## Nginx

### Instalación

```bash
apt update && apt install -y nginx
```

### Virtual host Grafana `/etc/nginx/sites-available/grafana`

```nginx
server {
    listen 80;
    server_name _;

    location / {
        proxy_pass http://10.1.1.66:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### Activar y reiniciar

```bash
ln -s /etc/nginx/sites-available/grafana /etc/nginx/sites-enabled/
rm /etc/nginx/sites-enabled/default
nginx -t && systemctl restart nginx
```

## Port forwarding en Proxmox

Para que el tráfico externo llegue al CT105 desde `192.168.3.200`:

```bash
iptables -t nat -A PREROUTING -i vmbr0 -p tcp --dport 80 -j DNAT --to-destination 10.1.1.35:80
netfilter-persistent save
```

### Flujo de acceso

```
Mac (192.168.3.105)
   ↓ http://192.168.3.200
Proxmox vmbr0 (NAT :80 → 10.1.1.35:80)
   ↓
CT105 Nginx (10.1.1.35:80)
   ↓ proxy_pass
CT101 Grafana (10.1.1.66:3000)
```
